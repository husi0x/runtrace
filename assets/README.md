# Assets

This folder is for README visuals and repository presentation assets.

Recommended hero image path:

```text
assets/runtrace-hero.png
```

Recommended size:

```text
1600x533
```

Any 3:1 banner ratio is fine.

Keep assets lightweight. Do not commit huge raw generation files, layered design files, or temporary exports. If a large source file is needed, keep it outside the repository and commit only the optimized final asset.

No fake images are included. Add `assets/runtrace-hero.png` only after a real hero image exists.
