# Runtrace examples

This folder contains sample output generated by Runtrace.

```text
examples/sample-output/
├── report.md
├── report.html
└── output.log
```

Open the HTML sample locally:

```bash
xdg-open examples/sample-output/report.html
```

On macOS:

```bash
open examples/sample-output/report.html
```

## Generate a fresh sample

From the repository root:

```bash
source .venv/bin/activate
runtrace demo
```

Runtrace prints the generated report paths. The files live under:

```text
.runtrace/runs/<run_id>/
```

The sample output is checked in only as a small example of the report format. Real run records stay local in `.runtrace/` and are ignored by git.
